About

  • All stats are accurate (taken from scraped data) on the date of last page update
  • Stats do not account for map collaborations: uploader gets all the numbers
  • Click table headers to sort the data; use Shift+Click for multi-column sorting, Ctrl+Click to reset the order (or just press the blue button)
  • Filter inputs support multiple comparison operators: read more and see examples here
    For example, enter >100 in Num column to leave only rows that have values greater than 100
    To find more than one name in Text column you can input: rayn*|ryg*
    For Date almost any format can be used, e.g. >=2/14/19 or <May 2019
  • Updates are manual and will be coming every once in a while
